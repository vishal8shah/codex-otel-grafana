Signal roles

Each store has a bounded job.

LOKI

Logs and derived records

Schema-confirmed raw event records are analyzer inputs. Clearly named derived records are dashboard inputs.

TEMPO

Trace context

Observed spans provide execution context. The shipped analyzers do not invent a join where safe same-run correlation is absent.

METRICS

Pipeline evidence

Collector health and spanmetrics help verify the pipeline. They are not native codex_* Prometheus metrics.

Grouping semantics

Useful aggregation without unsafe identifiers.

The absence of a confirmed safe request or call ID keeps several views deliberately aggregate-level.
