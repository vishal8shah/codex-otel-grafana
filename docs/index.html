Local-first. Privacy-safe. Evidence-led.

When Codex goes quiet, developers should not have to guess.

This diagnostic kit turns schema-confirmed Codex telemetry into focused local evidence. It targets the same classes of pain as stuck runs, MCP startup hangs, uncertain tool dispatch, stalled review or resume flows, backend reliability, and token ambiguity—without claiming unbuilt diagnostics or fixes for Codex bugs.
