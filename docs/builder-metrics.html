Issue-led diagnostics

Capabilities, not decorative promises

The kit targets real failure classes, but it promotes a diagnostic to shipped only after its source fields, privacy boundary, analyzer path, and dashboard behavior are proven. The interactive dashboard walkthrough shows how each shipped view supports a bounded investigation using reviewed synthetic example data.

Token economics is not token burn

Confirmed token fields on completed codex.sse_event records can support completed-run economics after a local run. This site does not provide fixed model rates, default cost estimates, or decorative dollar totals.

Token burn without completion is not a shipped diagnostic. It was removed because the required non-completion token shape was not confirmed in SCHEMA.md.

Codex Diagnostic Command Center

Symptom: Codex feels wrong, but I do not know which diagnostic to open first.

Meaning and action: Existing privacy-safe derived evidence is summarized by pain class. Open the linked focused dashboard and inspect the relevant grouping; treat it as investigation evidence, not proof of a Codex bug.
