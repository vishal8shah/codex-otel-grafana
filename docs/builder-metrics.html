Capabilities, not decorative promises

Token economics is not token burn

Confirmed token fields on completed codex.sse_event records can support completed-run economics after a local run. This site does not provide fixed model rates, default cost estimates, or decorative dollar totals.

Token burn without completion is not a shipped diagnostic. It was removed because the required non-completion token shape was not confirmed in SCHEMA.md.
